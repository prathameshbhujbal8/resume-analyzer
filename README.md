# AI Resume Analyzer (NLP Project)

This project analyzes how well a resume matches a job description using Natural Language Processing techniques.

## 🚀 Live Demo

https://resume-analyzer-vwwhywtuqiwauxgkk3bnc5.streamlit.app/

## 📌 Features

* Match score using TF-IDF and cosine similarity
* Skill matching and missing skill detection
* Suggestions to improve resume
* Interactive UI built with Streamlit

## 🧠 Tech Stack

* Python
* Scikit-learn
* Pandas
* Streamlit

## ⚙️ How it works

The system converts resume and job description into numerical vectors using TF-IDF. Cosine similarity is then used to calculate the match score. Skill-based comparison is also performed to improve interpretability.

## ▶️ Run Locally

1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   streamlit run app.py
